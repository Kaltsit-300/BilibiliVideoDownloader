##  核心优点 (Features)

与市面上常见的网页解析工具或抓包插件相比，本程序拥有以下压倒性优势：

*    **突破画质天花板，解锁真·超清**
    内置账号 Cookie 鉴权机制，彻底告别游客身份的 480P 模糊画质。最高可无损抓取 **1080P高码率、60帧 甚至 4K** 级别画质及无损音质（视您的B站账号等级而定）。
*    **独家“防发灰”色彩保护机制**
    市面工具大多默认抓取体积大且老旧的 AVC(H.264) 编码，这在处理特定黑白视频或全色域视频时会导致“纯黑变发灰”的严重 Bug。本程序采用**智能两轮淘汰算法**，强制优先抓取色彩 100% 准确、体积更小、细节更好的 **HEVC (H.265) / AV1** 编码格式。
*    **毫秒级无损混流封装**
    针对 B 站底层的 DASH 音视频分离机制，程序内置 FFmpeg 接口，下载完成后自动进行**免转码无损合并**，全程画质、音质零损耗。
*    **全自动零干预 & 桌面直达**
    只需输入视频链接，程序会自动爬取原视频标题进行重命名，并**跨平台智能识别您的电脑桌面路径（Win/Mac通用）**。合成好的视频直接保存到桌面，同时自动销毁所有临时文件，绝不制造系统垃圾。

## 🛠️ 安装与配置 (Installation)

### 1. 环境依赖
请确保您的电脑已安装 Python 3.7 或更高版本。打开命令行 (CMD) 或终端 (Terminal)，安装必备的第三方库：
```bash
pip install requests moviepy
```

### 2. 配置专属 Cookie（ 获取超清权限的钥匙）
*注：B站的 Cookie 存在时效性，如果某天您发现下载的视频变成了 480P，请执行此步骤更新 Cookie。*

1. 使用电脑浏览器（如 Chrome / Edge）登录您的 B 站账号。
2. 随意点开一个视频，按键盘 `F12` 打开“开发者工具”，点击 **Network (网络)** 标签页。
3. 按 `F5` 刷新网页，在左侧列表中随便点击一个请求，在右侧找到 **Request Headers (请求标头)**。
4. 找到 `cookie:` 这一行，将其后方**一长串字符全部复制**。
5. 用代码编辑器打开本 Python 脚本，找到代码开头的 `headers` 字典，将复制的内容粘贴到 `'cookie': "在此处粘贴您的Cookie"` 中（**请务必保留最外层的双引号**）。

##  极简使用方法 (Usage)

1. 双击运行该 Python 脚本，或在终端中执行：
   ```bash
   python bilibili_downloader.py
   ```
2. 在控制台的提示框中，**粘贴您想要下载的 B 站视频网址**（例如：`https://www.bilibili.com/video/BV1xx...`），按下回车。
3. 程序将自动进行嗅探、下载画面、下载音频并合成。
4. 提示“大功告成”后，**直接去您的电脑桌面**，完美的超清 MP4 视频已经准备就绪！

##  常见问题与观影提示 (FAQ)

**Q: 为什么下载下来的视频，用 Windows 自带的播放器打开只有声音没有画面？**

**A:** 这是因为本程序为了给您提供最顶级的画质和最正确的色彩，优先下载了最先进的 **HEVC (H.265)** 格式。Windows 自带的“电影与电视”播放器默认由于缺乏授权，不支持该解码。

 **完美解决方案**：强烈建议下载安装专业的万能本地播放器 **[PotPlayer](https://potplayer.daum.net/)** 或 **[VLC](https://www.videolan.org/)**。
